>

## 安装



```shell
# 设置国内镜像 可选
pip3 config set global.index-url https://mirrors.ustc.edu.cn/pypi/web/simple
# 安装依赖
pip3 install wxPytho

pip3 install send2trash

pip3 install time

```



## 运行



```python
 
   python main.py

```
